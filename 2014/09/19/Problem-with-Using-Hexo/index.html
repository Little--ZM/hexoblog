Problem with Using Hexo

使用hexo出现的一些问题

当我第一次使用hexo的时候 使用命令hexo init blog 却失败了，告诉我主题不存在

不断google之后解决问题

解决问题方法

命令hexo init blog 只能用在根目录下的目录 如 /data/webapps/
一旦在用户目下，就不行 如在 ~/hexo/ 就会出现theme不存在的问题
